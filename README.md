# 프로젝트 이름

- ZOOVIE

## 기능

- 사용자 등록 및 로그인 기능
- 영화 및 TV 프로그램에 대한 정보 제공
  1. 포스터
  2. 키워드
  3. 관련 영상
  4. 스틸컷
  5. 줄거리
  6. 주요 출연진
  7. 비슷한 영화
  8. 스트리밍 플랫폼
- 영화 및 TV 프로그램 북마크 기능
- 검색 기능 제공

## 기술 스택

- **서버**: Flask
- **데이터베이스**: MongoDB
- **클라이언트**: Dart/Flutter
